# Midterm Project – AI-Integrated IIoT Cybersecurity Plan

## Overview

This project focused on designing a cybersecurity strategy for a healthcare monitoring environment powered by AI-integrated Industrial IoT devices.

The system included wearable sensors, edge gateways, cloud analytics, and clinical dashboards used to monitor patient vitals in real time.

## Key Topics Covered

- Threat modeling
- Network segmentation
- Encryption (TLS 1.3 / AES-256)
- Role-Based Access Control (RBAC)
- HIPAA compliance
- Intrusion Detection Systems
- AI model poisoning defenses
- Penetration testing simulations

## What I Learned

- Security risks in connected healthcare systems
- Layered defense strategies
- AI-specific cyber threats
- Secure architecture planning
- Importance of continuous monitoring
