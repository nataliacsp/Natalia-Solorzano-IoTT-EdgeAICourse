# Capstone Project – Conceptual ICU Patient Monitoring Agent

## Overview

This capstone project proposed an autonomous ICU patient monitoring system that uses Edge AI and Generative AI to detect patient deterioration early and alert clinical staff in real time.

The solution was designed as a conceptual healthcare AI system focused on safety, explainability, low latency, and privacy.

## Core Technologies

- TimeGAN for synthetic ICU training data
- Autoencoder for anomaly detection
- Clinical Language Model for explainable alerts
- Edge AI deployment architecture
- Secure hospital network integration

## Goals

- Faster emergency alerts
- Reduced false alarms
- Improved patient monitoring
- Explainable AI decision support
- HIPAA-aligned architecture

## What I Learned

- Designing real-world AI systems
- Combining multiple AI models in one pipeline
- Healthcare workflow considerations
- Security and ethics in AI deployment
- Edge computing system design
